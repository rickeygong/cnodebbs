# CNODE BBS

## 前言

练手项目，模仿 CNODE 社区，接口来自[CNODE 社区](https://cnodejs.org/api)。

用到的知识点：

- html
- css
- javascript
- vue
  - v-on
  - v-bind
  - v-model
  - 组件
  - axios
  - vuex

## 基本架构

组件：

- 头部 Header
- 列表 PosltList
- 文章详情页 Article
- 侧边栏 SlideBar
- 用户个人信息 UserInfo
- 分页组件 Psgination
